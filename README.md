# 📊 Regional Sales Analysis | End-to-End Data Analytics Project

## 📌 Overview

This project demonstrates a complete **Data Analytics workflow** by analyzing five years of regional sales data. The objective is to uncover business insights, identify sales trends, evaluate regional performance, and build an interactive dashboard to support data-driven decision-making.

The project covers the entire analytics pipeline—from data loading and cleaning to exploratory data analysis (EDA), dashboard development, and business reporting.

---

## 📂 Dataset

The dataset contains historical sales records, including:

* Customer Information
* Product Details
* Sales Transactions
* Revenue & Profit
* Sales Channels
* Geographic Regions
* Budget Data

The data was collected from multiple tables and consolidated into a single analysis-ready dataset.

## 📈 Project Workflow

### 1. Data Loading

* Imported raw datasets into Python.
* Performed initial exploration of the data structure.

### 2. Data Cleaning

* Merged multiple datasets.
* Removed unnecessary columns.
* Standardized column names.
* Created new calculated fields.
* Prepared an analysis-ready dataset.

### 3. Exploratory Data Analysis (EDA)

Performed detailed analysis to understand:

* Sales trends over time
* Product performance
* Customer behavior
* Regional sales distribution
* Profitability analysis
* Sales channel comparison
* Correlation between numerical variables

Visualizations were created using Python to identify patterns and business opportunities.

### 4. Dashboard Development

Built an interactive **Power BI Dashboard** featuring:

* Sales Performance Overview
* Regional Analysis
* Customer Segmentation
* Product Performance
* Revenue Trends
* Profit Analysis
* Interactive Filters & Slicers

### 5. Business Report & Presentation

Prepared:

* Business Insights Report
* PowerPoint Presentation summarizing findings
* Actionable recommendations based on analysis

## 🚀 How to Run

1. Clone the repository.

```bash
git clone https://github.com/yourusername/Regional-Sales-Analysis.git
```

2. Install the required Python libraries.

```bash
pip install pandas numpy matplotlib seaborn
```

3. Open the Jupyter Notebook.

```bash
jupyter notebook
```

4. Run the notebook sequentially to perform:

   * Data Loading
   * Data Cleaning
   * Exploratory Data Analysis

5. Open the `.pbix` file using **Power BI Desktop** to explore the interactive dashboard.

6. Review the report and presentation for business insights and recommendations.
